Linaz - Built-in Linux Command A to Z

The game is simple. Think of a built-in unix command staring with the given letter. However, this is surprisingly hard since it seems we programmers don't index the command in this way as my friends and I have found out when playing this game.

How to play

Enter as many built-in unix command as you can think for every letter of the alphabet. For multiple entries, separate each by a comma. Click the "Start Game" button to play.
